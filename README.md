# 프로젝트명 : 발달장애 케어 시스템

## 프로젝트 개요
 - 기획 의도 : 사용자·기관 정보와 지원 이력을 통합 관리하고, 우선순위 기반 서비스를 제공하는 시스템 구축
 - 개발 동기 : 발달장애인에게 꼭 필요한 순간, 더 쉽게 도움을 제공하는 시스템 구축
 - 기대 효과 : 데이터 기반의 체계적인 지원 관리를 통해 발달 장애인들이 건강한 유대관계 형성 및 사회성 증진 발달장애인의 실질적인 삶의 질 향상

## 팀원
 - 고동현(팀장) - 서버 배포
 - 이태호(부팀장) - Git 관리
 - 김경환 - 개발 환경 구축
 - 김민지 - DB 관리
